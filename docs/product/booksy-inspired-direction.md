# Booksy-inspired direction — AgendaZap

## Posicionamento

AgendaZap é uma plataforma de agendamento e relacionamento para prestadores locais que atendem por horário, com link público simples para o cliente e painel completo para o negócio.

## O que absorvemos do Booksy

- Perfil público confiável, com serviços, localização, preço quando existir e avaliações seguras.
- Agendamento simples: abrir link, entender, escolher serviço, escolher horário e confirmar.
- Conta do cliente fácil de descobrir, com histórico e próximos atendimentos.
- Rebooking como continuidade natural do relacionamento.
- Avaliações como prova social, sem expor dados sensíveis.

## O que não copiamos agora

- Marketplace, busca global e ranking público.
- Portfólio/fotos, equipe/profissionais e agenda visual por profissional.
- Pagamento, sinal online, app nativo, notificações e WhatsApp Cloud API.

## Diferença estratégica

O AgendaZap começa como agenda online própria do prestador, não como diretório. A experiência é WhatsApp-first, brasileira, simples para pequenos negócios e flexível para segmentos além de beleza.

## Visão para cliente final

O cliente deve sentir que está lidando com uma vitrine real do prestador: entende o que é oferecido, escolhe o serviço, vê horários, confirma com conta simples e acompanha tudo em `/cliente`.

## Visão para prestador

O prestador opera agenda, serviços, clientes, histórico, avaliações e integrações sem depender de banco, seed ou suporte manual recorrente.

## Visão para admin

O Super Admin mantém a plataforma, planos, prestadores, assinaturas, templates e auditoria por interface protegida.

## Próximos módulos futuros

- Rebooking mais inteligente.
- Avaliações públicas com preferências de privacidade.
- Calendário visual.
- Profissionais/equipe.
- Portfólio/fotos.
- Ferramentas de divulgação WhatsApp.
- QR Code do link público.
- Relatórios de conversão e recorrência.
